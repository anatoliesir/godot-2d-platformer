# Little Godot 2D Platformer
A litltle and cute 2d platformer. It is unfinished, and I don't plan to finish it sooner or later.

If you want to test my game, I posted it on itch.io: 
https://anatoliesir.itch.io/godot-2d-platformer

Controls:\
A - left\
D - right\
Space - jump\

The assets used in this game are not mine.

This game teached me a lot about language that is like python and planning ahead of time. I Was able to learn to be more organised with my code and I could adapt to this environment pretty fast. This platformer was a challenge I couldn't avoid, because I am passionate about the new things I discover and create. And as you can see, this was a good opportunity to learn and adapt.
